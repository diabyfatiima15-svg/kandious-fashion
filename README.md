# 👗 KANDIOU'S Fashion — Système de Gestion de Boutique

> Projet de fin de formation FullStack — ODC Guinée 2026

## 📌 Description

KANDIOU'S Fashion est un système complet de gestion d'une boutique 
de vêtements féminins, développé dans le cadre de la formation 
FullStack à ODC Guinée.

L'application permet de gérer l'ensemble des opérations d'une 
boutique : clients, produits, achats, ventes, stocks, facturation 
et statistiques.

## 🛠️ Technologies utilisées

### Frontend
- React.js + Vite
- CSS3 (design Noir & Or)
- Chart.js (graphiques)
- jsPDF + QRCode (génération de factures PDF)
- XLSX (export Excel)

### Backend
- Java 17
- Spring Boot 3
- Spring Security + JWT
- Spring Data JPA + Hibernate
- MySQL
